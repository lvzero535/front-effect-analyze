# 项目修改影响分析

1. 用户给一个项目目录，该项目是一个Vue项目，项目代码包含vue，js，ts, css， less文件。
2. 当前用户修改了项目中的某个文件，需要分析该文件的影响范围，包括该文件直接或间接依赖的其他文件。

## Vue文件分析

1. 分析Vue文件的依赖关系，包括模板中的组件、指令、事件等。
2. 分析Vue文件的代码逻辑，包括组件的选项、方法、计算属性等。
3. 分析Vue文件的样式，包括组件的样式、scoped样式等。

## TS分析

1. 分析TS文件的依赖关系，包括导入的模块、导出的模块等。
2. 分析TS文件的代码逻辑，包括类型定义、函数、类等。
